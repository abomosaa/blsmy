# AI Development Guide

Project stack:
- Frontend: Angular 21
- Styling: Tailwind CSS
- Backend: Supabase

## Goal
Build a clean, fast, responsive website with simple UX and one main CTA per section.

## Default UX Rules
- Keep the UI simple
- Use one main CTA per section
- Clear navigation
- Fast loading
- Clean typography
- No clutter

## Allowed Tasks
- Generate pages
- Generate components
- Build forms
- Write Supabase queries
- Fix bugs
- Improve responsive design
- Refactor code

## Workflow
1. Create layout
2. Build Home page
3. Build remaining pages
4. Add forms
5. Connect Supabase
6. Improve responsive behavior
7. Test and clean up

AI MUST:
- Use the exact content
- Keep headings and text meaning unchanged

AI MAY:
- Improve formatting
- Adjust responsiveness

AI MUST NOT:
- Rewrite content
- Add fake text
- Add missing sections

apply_patch-test: OK
